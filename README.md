# Study Schedule 📚✨

A modern and aesthetic study schedule website designed to help students organize their weekly study plans and track their progress.

## Features 🚀

- Weekly study schedule pages
- Interactive checkboxes
- Progress automatically saved using JavaScript and localStorage
- Responsive design for phones and computers
- Smooth navigation between weeks
- Soft aesthetic gradient UI
- WhatsApp contact button

## Built With 💻

- HTML
- CSS
- JavaScript
